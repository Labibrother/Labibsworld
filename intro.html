Coding and stuff

I've always seen kids younger than me go to hackathons, olympiads, competetive programing but I was afraid of coding. Unless one of my teacher taught me C and I found out it was not that scary as I expected. When I first started learning HTML and CSS, it was fun, but the more you learn the worse it becomes. Yabai! But when I was introduced to JavaScript, I was surprised to find that it is very similar to C.


Some tips

I'm still figuring out how all these web desinging stuff work. But I have found W3Schools more helpful than any other sources. No sign-up required, just start learning right away. I also watch Bro Code, but sometimes he covers a topic too fast of too briefly. For JavaScript, W3Schools has an example based learning method which is more effective I guess. After learning the basics, you should start playing around with stuffs you just learned- find a template, open devtools, edit it, break it!
